# Claude Mem

🧠 Persistent context across sessions for AI agents.

Claude Mem is described as a tool for preserving session context for agents by capturing what an agent does during sessions, compressing that information with AI, and bringing relevant context back into future sessions. It is presented as working with Claude Code, OpenClaw, Codex, Gemini, Hermes, Copilot, OpenCode, and more.

## Overview

Long-running agent workflows often span multiple sessions. Claude Mem is positioned around that problem: it keeps track of session activity, summarizes it with AI, and reuses relevant context later so future sessions can continue with better continuity.

Based on the available description, the project focuses on three core ideas:

- capturing agent activity during sessions
- compressing that history with AI
- injecting relevant context into later sessions

It is also described as being usable with multiple agent environments and coding assistants rather than being limited to a single platform.
